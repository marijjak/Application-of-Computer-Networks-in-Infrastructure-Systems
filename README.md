# Repozitorijum datoteka ľ PRMUIS projekat
TEMA: Sistem za upravljanje datotekama

Projekat implementira klijentľserver aplikaciju u C# jeziku.

## Funkcionalnosti
- UDP prijava klijenta
- TCP komunikacija za rad sa datotekama
- Citanje, izmena/kreiranje i brisanje datoteka
- Zakljucavanje datoteka tokom izmene
- Red cekanja zahteva
- Statistika po klijentu

## Struktura
- Biblioteka ľ zajednicke klase
- RepoServer ľ server (UDP + TCP)
- RepoClient ľ klijent aplikacija

## Tehnologije
- C#
- TCP / UDP soketi
- .NET Framework
